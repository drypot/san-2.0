   들어가는 글
   -----------

   산은 텍스트 에디터입니다.
   산은 공개 프로그램 (프리웨어) 입니다.


   동작 환경
   ---------

   CPU   :  386 이상, 486 시스템을 권장

   RAM   :  최소 2메가 Extended or XMS Memory
            4 메가 이상 권장

   VIDEO :  VGA 640x480, Vesa Local bus VGA 권장

   OS    :  도스 5.0 이상


   설치 사용 방법
   --------------

   꾸러미 안의 USING.TXT 화일을 참고해 주십시요.


   특징
   ----

   32bit CPU 를 활용합니다. 산은 32bit 코드로 만들어 졌습니다. 386
   이상의 32bit CPU 상에서 작동가능하며 도스상에서 32bit CPU의 기능을
   최대한 이용합니다.

   최대한의 유연성을 제공합니다. 설치되어 있는 메모리를 모두 사용하며,
   메모리 제한 안에서, 동시 편집 문서수, 화일 크기, 줄의 크기등에 대한
   제한이 없습니다. 물리적인 메모리가 부족할 경우, 하드 디스크를 임시
   저장영역으로 사용할 수 있습니다. 단 최대 가상 메모리 크기는 32MB 로
   제한됩니다.

   수백개의 화일을 동시에 불러 놓고 편집할 수 있습니다. 각 화일간에
   쉽게 자료를 이동할 수 있습니다. 한 화일은 한번에 수십만줄의 자료를
   가지고 있을 수 있습니다. 몇 백 킬로바이트의 문자열이라도 한줄에
   들어올 수 있습니다.

   충분한 기본기능을 제공합니다. 많이 사용되는 기본 이동, 편집 기능을
   대부분 수용하며, 사용되는 위치에 따라서 적절한 처리를 하도록
   보강되었습니다.

   일반 텝 문자 처리 기능 외에, 윗줄의 공백을 따르는 스마트 텝기능을
   제공합니다. 텝 문자 크기는 사용자 임의로 설정하실 수 있습니다.

   한글 두벌식, 한글 세벌식, 알파벳 쿼티, 알파벳 드보락 자판을
   지원합니다.

   문서의 코드 형태, 즉 조합형, 완성형, 정상 조합형을 자동 인식하며,
   읽어 들여온 문서의 코드 변환이 쉽습니다. KS5061C 에 정의 되어 있는
   특수 문자와 한자를 완벽히 지원합니다.

   국내 제작 에디터에서는 처음으로 다단계 UNDO/REDO 기능을 제공합니다.
   UNDO Buffer 의 크기를 사용자 임의로 설정할 수 있으며, 버퍼가
   충분하다면 몇시간 동안 수정한 화일의 내용을 원상 복귀할 수
   있습니다.

   영역 블럭, 행 블럭 기능을 제공합니다. 산 1.2 판과 달리, 블럭 정렬
   기능이 글쇠 한번에 가능합니다. 즉, 블럭 설정, 블럭 정렬, 다음 문단
   처음으로 이동하는 단계가 한번의 글쇠 동작으로 가능합니다.

   블럭의 인덴테이션도 한번의 글쇠 동작으로 가능합니다. 공백삽입
   방식의 블럭 인덴테이션과 텝 문자 삽입 방식을 제공합니다.

   도스 쉘 기능과, 외부 화일 실행 기능을 제공합니다. 외부 화일 실행시
   현재 편집중인 화일의 드라이브, 디렉토리, 이름, 익스텐션, 패스
   정보등을 전달할 수 있습니다. 도스 쉘 이용시 충분한 기본 메모리를
   돌려드립니다.

   산은 빠릅니다. 대충 구현한 광고성 기능은 가지고 있지 않습니다. 한번
   지원하는 기능은 확실하고 막강하고 효율적입니다.


   주의 사항
   ---------

   이 프로그램에 의한 피해에 대해서 제작자에게 보상을 요구할 수
   없습니다.

   생산적인 작업을 하시기 전에, 자신이 사용하는 시스템, 소프트웨어와
   본 프로그램이 충돌없이 작동하는지 시험하십시요.

   상업적 목적의 배포를 금하며, 허락 없이 다른 물건에 끼워서 배포하는
   것을 불허합니다. 피치 못할 경우, 제작자에게 미리 연락하시기
   바랍니다.

   프로그램 배포에 직접적으로 가격을 메기는 것을 불허합니다.

   이 소프트웨어는 제작자가 배포한 모든 화일이 포함된 형태로만 배포될
   수 있으며, 압축 형식은 바뀔 수 있지만, 다른 프로그램과 섞여서는
   안됩니다.

   본 프로그램은 공개 프로그램 (프리웨어) 입니다. 위 사항이 지켜진다는
   조건 하에 프로그램은 자유롭게 배포, 사용될 수 있습니다.


   사용시 주의 사항
   ----------------

   제작시 프로그램의 오류를 막기 위해서 최대한 노력하였습니다만,
   여건상 충분한 환경에서 실험하지 못했습니다.

   산을 시험 운행하시기 전에 노턴 유틸리티의 image나 같은 기능을 하는
   프로그램을 이용해서 하드 디스크의 주요 테이블 정보를 복사해 두시기
   바랍니다. 사용자가 입력한 데이턲등 쉽게 다시 제작할 수 없는 화일은
   별도로 보존하시기 바랍니다.
