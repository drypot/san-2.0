
   산의 앞으로의 개발 계획

   1994.10.1   10.16

   박규현

------------------------------


   들어가는 글
┍━━━━━━━
│ 91 년 이후 산을 공개하기 시작한지 4 년이 흘렀습니다. 그중 2  년간은
│ 꾸준히 판올림을 했었으나, 이후 2 년간은 몇가지 이유로 해서 실질적인
│ 판올림은 없었습니다.


   산 1.X 판
┍━━━━━━
│ 1994.9.25 일 기준으로, 그동안 쉐어웨어로 공개되었던 산  1.2.X  판과
│ 그 이하판은 프리웨어화 했고, 소스 코드도 공개했습니다. 이것은  산의
│ 공개 초기부터 사용되었던 형식의 코드로 C 와  8086  어셈블리  언어로
│ 작성되어 있습니다.
│
│ 본인이 제공하는 1.X  판은  1.2.1  이  마지막  이지만,  소스  코드를
│ 공개했으므로, 산 1.X 판의 개정 작업이 다른 개발자들에 의해서 이루어
│ 질 수 있게 되었습니다.


   산 2.X 판
┍━━━━━━
│ 기존의 코드는 앞으로의 기능 개선에 몇가지  중요한  문제점을  가지고
│ 있기 때문에, 93 년부터 틈틈이 프로그램을  다시  설계하고  있습니다.
│ 이것은 1.X 판과는 전혀 다른 프로그램이지만 판수를 달리하여 프로그램
│ 이름을 계속 산으로 하기로 했습니다.
│
│ 현재 C++ 3.0 과 8086 어셈블리를 사욤해서  작업하고  있고,  일차적인
│ 목표는 메모리관리 개선, 매크로 언어 지원, 개선된 Undo/Redo,  그리고
│ Marking 기능을 꼽고 있습니다.
│
│ 이미 메모리 관리 부분이 사용할 수 있게 되어, 기본적인 기능만 장착된
│ 산 2.0.X 판을 공개했습니다. 이 판은 그동안  산의  판올림이  2  년간
│ 없었던 관계로 좀더 빨리 새 판을 공개해야 겠다는  제  조급한  마음에
│ 이루어져 아직 미진하지만, 앞으로 기능을 계속 추가시킬 것입니다. 2.X
│ 판은 다양한 환경의 지원을 계획하고 있습니다.


   나가는 글
┍━━━━━━
│ 요약하자면,
│
│ 프리웨어로 공개된 1.X 판에는 더이상 본인의 지원은 없습니다.
│
│ 앞으로는 2.X  판을  쉐어웨어로  공개하면서,  이것의  개발에  전념할
│ 것이며, 아직 1.X 판에 비해  기능이  떨어지지만,  개발  계획에  따라
│ 기능을 단계적으로 강화시킬 것입니다.
│
│ 산보다 훨씬 강력한 영문, 한글 에디터들이 존재하는 것이  사실입니다.
│ 하지만, 이에 상관 없이, 제 나름대로의 꾸준히 밀고 나갈 것입니다.
