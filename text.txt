   1996.05.07
   박규현
------------------------------------

   들어가는 글
   -----------

   텍스트서비스(TextService)는 텍스트 에디터(Text Editor)입니다.


   실행방법
   --------

   binos2/t.exe 를 실행시키십시요.


   주의 사항
   ---------

   제작자는 이 소프트웨어의 정상 작동에 대한 보증을 하지 않습니다.
   제작자는 이 소프트웨어에 의한 피해에 대해 배상할 의무를 지지 않습니다.
   생산적인 작업을 하시기 전에, 자신이 사용하는 하드웨어, 소프트웨어와
   본 프로그램이 충돌없이 작동하는지 시험하십시요.

   이 프로그램은 프리웨어입니다.


   지원 상황
   ---------

   [제한]      64MB 제한 안에서, 열수 있는 화일수, 화일의 크기, 행의 크기제한이
               없습니다.

   [자판]      문자 배열은
               한글 두벌식, 한글 세벌식, 알파벳 쿼티, 알파벳 드보락을
               지원합니다.

   [코드]      (KS5601C 조합형, 완성형) 과 정상 조합형을 지원합니다.
               문서가 어떤 형식의 코드로 만들어 졌는지는 텍스가 판단합니다.

   [하드웨어]  출력은 VGA 에서만 가능합니다.
               키보드는 101/103 키보드만 지원합니다.

   [운영체제]  OS/2, DOS 에서 동작하는 바이너리 생산이 가능합니다.

               OS/2 버젼은 32비트 코드로 만들어졌으므로 2.X 이상에서만
               동작합니다.

               이 프로그램은 Fullscreen 용입니다.


   기술적 사항
   -----------

   DOS Version 과 OS/2 Version 사이에 거의 모든 소스코드를 공유하며,
   platform independent 합니다.

   버젼업이 된다면, DOS 용 바이너리와 OS/2 용 바이너리가 같은 시점에
   나올 수 있습니다.

   제데로된 C++ Complier 가 있고, 부수적인 지원이 있다면, 큰 어려움 없이
   다른 platform으로 이식이 가능할 것입니다.
   한글 에디터가 필요하고, 개발 지원 의사가 있는 분은 연락주십쇼.

   DOS Version 의 경우, 드라이버들이 Assembly Language 로 만들어 졌으나,
   OS/2 Version 은 모두 C++ 로 만들었습니다.

   텍스 OS/2 Version 의 Source Code 는 키보드 모듈과 스크린 출력 모듈을
   포함해서 모두 C++ 로 쓰여 있습니다.


   사용한 도구
   -----------

   OS/2 Warp 영문판
   The Semware Editor Junior for OS/2 4.0 (QEdit)
   Watcom C/C++ 10.0b
   Borland C/C++ for OS/2 2.0
   Borland C/C++ for DOS & Windows 4.0
   Turbo Assembler


   자판 설정
   ---------

   etc₩t.cfg 화일을 수정합니다.

   초기치는 두벌식과, 쿼티자판입니다.

   세벌식 한글 자판 사용자나, 드보락 영문자판 사용자는
   다음과 같이 두벌식, 쿼티 지정행을 주석화 해서 생략시키시고,
   세벌식, 드보락 지정행의 주석 표시를 없애시기 바랍니다.

   예)

   초기 형태

   meta_code_page          -₩mcpage₩qwerty.mcp
   #meta_code_page         -₩mcpage₩dvorak.mcp
   meta_code_page          -₩mcpage₩han3-90.mcp
   #meta_code_page         -₩mcpage₩han2.mcp

   세벌식, 드보락 자판으로 수정된 형태

   #meta_code_page          -₩mcpage₩qwerty.mcp
   meta_code_page         -₩mcpage₩dvorak.mcp
   #meta_code_page          -₩mcpage₩han3-90.mcp
   meta_code_page         -₩mcpage₩han2.mcp

   Bugs
   ----

   검색/치환 기능이 제공되지 않습니다.
   선 그리기 기능이 제공되지 않습니다.
   컬럼 블럭 기능이 제공되지 않습니다.

   OS/2 에서 EA (Extended Attribute) 에 대한 지원이 없습니다.

   현재 판에는 Macro Compiler 가 같이 제공되지 않습니다.
   그러므로, 글쇠 배당을 바꾸 실 수가 없습니다.

   화일 선택 기능이 없습니다.

   문서의 코드 종류 판별시 시료가 적으면 다른 코드로 오인할 수 있습니다.
   수동으로 코드종류를 지정할 수 없습니다.

   설명서가 없습니다.
